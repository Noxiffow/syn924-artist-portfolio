# Artist Portfolio

Portfolio website built for a freelance digital illustrator.

## Stack

- [Astro](https://astro.build) (static)
- TypeScript
- CSS (OKLCH color system)
- Vercel

## Features

- Masonry gallery with content filter toggle
- Desktop lightbox
- Security headers (CSP, X-Frame-Options, etc.)
- Responsive design

## Dev

```sh
npm install
npm run dev
npm run build
```
